# Student List Management

## Overview

This repository contains educational C++ implementations of student management systems based on linked list data structures.

The projects demonstrate the creation, modification, traversal, and deletion of student records using both linear and circular linked lists.

## Projects

### 1. Linear Linked List of Students

Implementation of a student database using a singly linked list.

#### Student Information

Each record stores:

* Name
* Age
* Average Grade

#### Features

* Add a student to the end of the list
* Insert a student at a specified position
* Display all students
* Delete a selected student
* Delete the entire list
* Dynamic memory allocation for all nodes

#### Concepts Demonstrated

* Singly linked lists
* Recursive traversal
* Dynamic memory management
* Node insertion and deletion

---

### 2. Circular Doubly Linked List of Students

Implementation of a student database using a circular doubly linked list.

#### Student Information

Each record stores:

* Name
* Age
* Average Grade

#### Features

* Add a student to the end of the list
* Insert a student at a specified position
* Replace an existing student record
* Delete a selected student
* Delete all students
* Display all students
* Count the number of students
* Circular navigation through the list

#### Concepts Demonstrated

* Circular linked lists
* Doubly linked lists
* Bidirectional navigation
* Dynamic memory management
* Advanced insertion and deletion algorithms

## Technologies
